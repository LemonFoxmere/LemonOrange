DeepFusion

Neural Networking, Simplified

Watch Video

A Free AI Playground

DeepFusion is an online application that allows you to create and train neural networks without writing code. You can try out the program if you want to, or check out the GitHub repo here.

This project is still being developed!

AIVIS

A New Window For Neural Networks

AI, Visualized.

The predecessor to DeepFusion, AIVIS provides pretty much the same functionality to DeepFusion, just simplified even more. You get an actual representation of a MLP network, and can train it with custom data. Try out the program out, or check out the GitHub repo here.
